# Object & reflexive pronouns

The reflexive form is used when the object refers back to the subject. It differs from the plain object form only in the third person, where it is **sig**.

| Subject | Object | Reflexive |
|---|---|---|
| **jag** | **mig** | **mig** |
| **du** | **dig** | **dig** |
| **han** | **honom** | **sig** |
| **hon** | **henne** | **sig** |
| **den / det** | **den / det** | **sig** |
| **vi** | **oss** | **oss** |
| **ni** | **er** | **er** |
| **de** | **dem** | **sig** |
