# Noun inflection

Swedish nouns inflect along two axes: number (singular/plural) and definiteness. Definiteness is marked with a **suffix**, not a separate word.

|  | Indefinite | Definite |
|---|---|---|
| Singular | **en bok** | **bok`en`** |
| Plural | **böcker** | **böcker`na`** |

## The 5 declension classes

The plural suffix depends on the declension class of the noun.

| Class | Plural suffix | Gender | Example (sg. / pl., indef. → def.) |
|---|---|---|---|
| 1 | `-or` | en | **flicka** → **flickan** · **flickor** → **flickorna** |
| 2 | `-ar` | en | **bil** → **bilen** · **bilar** → **bilarna** |
| 3 | `-er` | en/ett | **natt** → **natten** · **nätter** → **nätterna** |
| 4 | `-n` | ett | **äpple** → **äpplet** · **äpplen** → **äpplena** |
| 5 | `-Ø` (none) | en/ett | **läkare** → **läkaren** · **läkare** → **läkarna** |

Second class-5 example: **brev** → **brevet** · **brev** → **breven**.

## Rules for determining the class

| Pattern | Examples | Class |
|---|---|---|
| en-word ending in unstressed `-a` | **flicka**, **kvinna** | 1 |
| ett-word ending in a vowel | **äpple** | 4 |
| ett-word ending in a consonant | **hus**, **barn**, **tåg** | 5 |
| en-word ending in `-are` | **lärare**, **skådespelare** | 5 |
